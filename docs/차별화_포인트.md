# 차별화 포인트 (면접 대비)

1. **RAG 심화:** 하이브리드 검색 + 리랭킹 + 의도별 검색 소스 최적화 + 한국어 형태소 분석
2. **게임 특화:** 호감도/감정/퀘스트 연동으로 "게임에 실제 적용 가능한" 수준
3. **스포일러 방지:** 퀘스트 진행도에 따른 힌트 수준 조절 → 게임 기획 이해도
4. **페르소나 일관성:** NPC마다 확실히 다른 성격/말투 유지 + 정량적 평가
5. **할루시네이션 관리:** 세계관 외 정보 생성을 탐지하고 측정하는 평가 체계
6. **보안:** 3단계 프롬프트 인젝션 방어 + 캐릭터 유지 거부 응답 + 방어율 정량 측정
7. **NPC 간 정보 전달:** Cross-NPC Memory로 살아있는 세계관 구현
8. **비용 최적화:** Semantic Cache로 LLM 호출 절감 + 의도별 검색 범위 제한
9. **StoryProof 연장선:** 기존 RAG 경험을 게임 도메인으로 확장했다는 스토리
